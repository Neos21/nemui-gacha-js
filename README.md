# nemu-gacha-js : ねむいガチャ.js

__[Enter This Website.](https://Neos21.github.io/nemui-gacha-js/)__


## How To Use on CLI

```sh
# `git clone` this repository, then `cd` the repository directory.
$ npm install
$ npm run cli 'ねむい'
# --> ねむい or ねむくない

# Force parse negative
$ npm run cli -- --force 'ねむい'
# --> ねむくない
```
